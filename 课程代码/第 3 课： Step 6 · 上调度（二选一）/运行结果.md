## B. crontab —— 本机可验证

cron 行用的是标准 5 字段格式：

```text
0 3 * * * cd ~/loop-demo && ./loop.sh >> ~/loop.log 2>&1
│ │ │ │ │
│ │ │ │ └─ 星期几 (0-7)
│ │ │ └─── 月   (1-12)
│ │ └───── 日   (1-31)
│ └─────── 时   (0-23)  ← 3 点
└───────── 分   (0-59)  ← 0 分
```

安装与验证（不破坏你现有 crontab，建议先 `crontab -l` 备份）：

```bash
crontab -l > my_cron_backup.txt   # 备份现有
crontab -e                        # 手动粘贴 crontab.txt 里那一行
crontab -l                        # 应能看到刚加的行
```

> 本目录**没有**自动改你的真实 crontab（那是对你系统的副作用，需你自己确认）。
> `crontab.txt` 已写好，按上面三步即可装上。
