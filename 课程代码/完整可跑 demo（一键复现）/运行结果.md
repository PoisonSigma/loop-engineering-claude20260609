# 运行结果 · 完整可跑 demo（一键复现）

一条命令把第 3 课四件套从零跑通（不需要 claude CLI、不烧 token）：

```bash
bash setup_and_run.sh
```

## 三个关键验证点

1. **门一开始真能判失败**：Step 2 pytest `exit=1` + ruff format `exit=1`（除零 bug + 格式问题都被抓到）。
2. **循环靠门的退出码停**：Step 5 第 1 轮门变绿 → `✅ 门通过，停止` → `exit 0`，不是 Agent 自说自话。
3. **没作弊（只改 src/，没碰测试）**：收尾 `git diff --stat` 只列出 `STATE.md` + `src/calc.py`，
   **`test_calc.py` 没出现** —— 契约「禁碰 test_*.py」被遵守。

## 与真实 `claude -p` 的唯一差别

本脚本用 stub 假冒 Agent 的「修 src/」动作；换成真 `claude -p`（你机器装好 claude CLI 后），
`loop.sh` 其余逻辑（轮次 / 硬门 / 硬停 / 最小授权）完全不变。即把 `loop.sh` 里的 stub 路径去掉、
直接 `./loop.sh` 即可。
